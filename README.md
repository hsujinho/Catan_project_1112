## 遊戲指南

此遊戲的運作模式使用 SDL 作為地圖顯示的介面，並使用終端機做為遊戲的操作介面，輸入各種資料以進行遊戲。

### 遊戲開始

遊戲開始後，玩家將作為編號為 1 的玩家進行遊戲。而編號 2, 3, 4 的玩家為電腦玩家，其操作為電腦自動進行。

各玩家編號及其對應的顏色如下：
| 玩家編號 | 顏色 |
| :--------: | :----: |
| 1 | 紅色 |
| 2 | 白色 |
| 3 | 橘色 |
| 4 | 藍色 |
